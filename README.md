# 🔥 Valhalla Exhausts — Corporate Landing Page

## Overview

**Valhalla Exhausts** is a premium brand specializing in handcrafted custom exhaust systems, exhaust repair, and tailored automotive solutions. This landing page is designed to communicate quality, craftsmanship, and trust — while driving direct customer inquiries.

## Sections

### 1. Hero
Full-viewport opening with a strong brand headline, a supporting tagline, and two clear CTAs: *Request a Build* and *See Our Process*.

### 2. Services
Four core service cards — Custom Fabrication, System Repair, Performance Upgrades, and Restoration — presented in a scannable grid.

### 3. Process
A four-step visual timeline: Consultation → Assessment → Fabrication → Delivery. Transparent and trust-building.

### 4. Benefits
Side-by-side layout combining key differentiators (bespoke builds, fast turnaround, guaranteed workmanship) with social proof stats and a customer quote.

### 5. FAQ
Grid of the most common questions, covering vehicle compatibility, lead times, materials, pricing, and geographic coverage.

### 6. Contact
Direct contact channels (phone, WhatsApp, email) alongside a lightweight inquiry form — low friction, high intent.

## Customization

All brand colors and typography are defined as CSS variables at the top of the stylesheet:

```css
:root {
  --ember:   #c8441a;   /* Brand accent */
  --black:   #0a0a0a;   /* Background */
  --white:   #f5f4f0;   /* Primary text */
  --ash:     #888888;   /* Muted text */
}
```
